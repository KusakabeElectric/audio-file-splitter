# 音声ファイル分割ツール

100MB以上の音声ファイルを自動的に100MB以下に分割するWebアプリケーションです。

## 機能

- ドラッグ&ドロップまたはファイル選択で音声ファイルをアップロード
- 100MB以上のファイルを自動的に95MB以下に分割（MP3フレーム境界を考慮）
- 各分割ファイルを個別にダウンロード可能
- ブラウザ上で完結するため、サーバーにファイルをアップロードする必要がありません

## 使い方

1. `index.html`をブラウザで開く
2. 音声ファイルをドラッグ&ドロップするか、「ファイルを選択」ボタンから選択
3. 100MBを超えるファイルは自動的に分割されます
4. 各ファイルの「ダウンロード」ボタンをクリックして保存

## Web公開方法

### 方法1: GitHub Pages（無料・簡単）

1. GitHubアカウントを作成（まだの場合）
2. 新しいリポジトリを作成
3. このフォルダ内のファイル（`index.html`, `style.css`, `script.js`）をアップロード
4. リポジトリの Settings → Pages に移動
5. Source を "main" ブランチに設定
6. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]` でアクセス可能

### 方法2: Netlify（無料・簡単）

1. [Netlify](https://www.netlify.com/)にアカウント作成
2. このフォルダをドラッグ&ドロップ
3. 自動的にデプロイされ、URLが発行されます

### 方法3: Vercel（無料・簡単）

1. [Vercel](https://vercel.com/)にアカウント作成
2. GitHubと連携するか、フォルダをアップロード
3. 自動的にデプロイされます

### 方法4: その他の静的ホスティング

- Firebase Hosting
- Cloudflare Pages
- AWS S3 + CloudFront
- その他、静的ファイルをホスティングできるサービス

## 技術仕様

- 純粋なHTML/CSS/JavaScript（フレームワーク不要）
- ブラウザのFile APIを使用
- MP3ファイルのフレーム境界を検出して分割
- 外部ライブラリ不要

## 注意事項

- すべての処理はブラウザ上で実行されます
- ファイルはサーバーにアップロードされません
- 大きなファイルの処理には時間がかかる場合があります
- ブラウザのメモリ制限により、非常に大きなファイル（数GB以上）は処理できない場合があります

## ライセンス

自由に使用・改変・配布可能です。
